About this project

Mapping time
through design

This database collects and organizes design artifacts — objects, interfaces, systems, and images — that engage with the representation, manipulation, or experience of time. Cases span over two millennia, from ancient Greece to contemporary smart wearables.

Each entry is catalogued across multiple analytical dimensions: the domain and type of the artifact, its technological condition, the sensory modality it addresses, and the specific temporal operations it performs.

Data Fields

  • Case Name Title of the artifact or system
  • Author / Producer Designer, firm, or institution
  • Date Year of creation or publication
  • Artifact Domain Object, image, 2D/3D digital interface…
  • Artifact Type Calendar, wristwatch, game, platform…
  • Temporal Focus Category of temporal operation
  • Temporal Operation Signaling, organizing, repeating…
  • Representation of Time How time is shown or encoded
  • Temporal Orientation Past, present, future or combinations
  • Design Intent Awareness, optimization, estimation…
  • Design Solution The design approach adopted
  • Sensory Modality Vision, touch, sound…
  • Technological Condition Analog or digital
  • Interaction Mode Static, dynamic, interactive

How to use

Table shows all fields, is sortable by column, and supports multi-filter. Case Studies gives a visual overview. Data Viz shows distributions across all categorical dimensions. Alluvial lets you explore relational flows between any two dimensions.

Click any tag in Table or Cards to filter by that value. Case titles link to their original source.

Technical notes

Single-file static HTML — no server or database required. Data is embedded in the JavaScript DATA array. Charts use Chart.js; the alluvial diagram uses D3.js.
